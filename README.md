# GGGuide - Training project

_A little project just for training and learning infrastructure, APIs and full-stack development._

## Important note for this repo

This is a training project built quickly by reusing components and patterns from previous projects i do.
The main focus is on learning infrastructure setup, containerization with Docker and Docker Swarm orchestration.

So yes i take more time on infra than other stack for trainging to future project :p

## Stack

- **Front** -> React, Vite and TS (reused from previous projects)
- **Back** -> Hono, PostgreSQL and TS
- **API** -> REST
- **Infrastructure** -> Docker, Docker Swarm, Traefik, Nginx

## Overview

A website where anyone can create and share guides for their favorite games.
All the data is stored in a database and managed through an API.
